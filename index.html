数位dp的理解E. Salazar Slytherin's Locket

Problem - E - Codeforces

Des

q次询问，1e5级别，询问某个区间，b进制下每个数0-b-1出现的次数都为偶数次的个数。

Solution

很容易想到数位dp， 我们用一个数，状压一下维护进制位出现的次数，用^实现，关键是这里有一点很疑惑，之前都是无脑怼状态上去算就行，这里出现了t组，还是会重复查询的那种，反复memset容易超时，之前的写法都不太正宗。

其实我们只需要关心pos，status这两个位置就行，关于lead和limit不需要定义在状态中去搜索。这样应对多组的时候就不用反复清空数组。

离线+状压+数位dp

中国石油大学校赛题解

A

双指针

B

惨案现场，被卡精了。

C

n个数，求出所有子区间的最小公倍数的质因子个数总和。设两个数的lcm为x，则x包含了这两个数字的因子，包含质因子。oN的去处理当然不行，方向定在处理质因子的贡献，很显然，如过ai有质因子z，那么包含ai的子数组的lcm会含有质因子x，如果区间长度为len，那么z的贡献次数为len * (len + 1) / 2，还有一个问题，z会被重复计算贡献。那么怎么去掉重复贡献，假设ai a两个数都有质因子z，他们中间的数，都没有质因子z，那么减掉中间这段区间长度的贡献即可，因为重复贡献了这段区间两次。做法，筛出每个数字每个质因子的位置（质因数分解），然后遍历这些位置，减掉重复贡献。

D

签到

J

贝叶斯公式（离散数学。。。

不选赢的概率：1 / n

选赢的概率：1 / n * 0 （你选了对的然后去选不对的）+ (n - 1) / n * (1 / (n - 2)) （错里面选到对的概率）

显然2概率大，代码简单（看来又得复习点简单的概率论）

E

给你n堆石头，告诉你每一堆石头的数量，每块石头都是1重量，只有某一块是2，你每次可以提问m堆石头，它会给你返回这些堆石头的总重量，很显然，假设某个区间[L, R]，s为数量的前缀和，如果s[R] - s[L - 1] != query(L, R)那么2重量必然在这一堆里面，很显然的二分。

01 MST

Problem - D - Codeforces (Unofficial mirror by Menci)

Des

给你一个完全图，边的权值不是1就是0，给出那些权值为1的点，求MST。

Solution

直接正向MST算法，不行，边太多了，我们维护n棵生成树值为0的连通块个数，最后答案为权值为0连通块的个数-1（n个点最小生成树为m - 1条边），问题是怎么维护？对于某个点，假设现在有一堆权值为0连通块，是不是存在一个结论。如果这点能加入这个连通块，这个点到该块权值为1的边数小于连通块的点数。

[传送门](E-E-相约桂子山_华中师范大学2022级程序设计新生赛（同步赛） (nowcoder.com))

给你一个完全图，问有多少种方式，遍历1-n所有点再回到1，每个方案使用的边都不能有一条相同。

Solution

性质题还是太难了，因为完全图每个点度n - 1，那么我们分一半当出边，一边当入边，那么直接输出
