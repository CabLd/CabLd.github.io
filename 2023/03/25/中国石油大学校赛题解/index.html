中国石油大学校赛题解

A

双指针

B

惨案现场，被卡精了。

C

n个数，求出所有子区间的最小公倍数的质因子个数总和。设两个数的lcm为x，则x包含了这两个数字的因子，包含质因子。oN的去处理当然不行，方向定在处理质因子的贡献，很显然，如过ai有质因子z，那么包含ai的子数组的lcm会含有质因子x，如果区间长度为len，那么z的贡献次数为len * (len + 1) / 2，还有一个问题，z会被重复计算贡献。那么怎么去掉重复贡献，假设ai a两个数都有质因子z，他们中间的数，都没有质因子z，那么减掉中间这段区间长度的贡献即可，因为重复贡献了这段区间两次。做法，筛出每个数字每个质因子的位置（质因数分解），然后遍历这些位置，减掉重复贡献。

D

签到

J

贝叶斯公式（离散数学。。。

不选赢的概率：1 / n

选赢的概率：1 / n * 0 （你选了对的然后去选不对的）+ (n - 1) / n * (1 / (n - 2)) （错里面选到对的概率）

显然2概率大，代码简单（看来又得复习点简单的概率论）
